¿Qué elementos se tomaron en cuenta para hacer el análisis?

Se utilizó la teledetección la cual es la ciencia que se encarga de obtener, almacenar y procesar información satelital. Esto nos permite hacer análisis a pequeña y gran escala de manera extremadamente rápida. Además, se utilizaron redes neuronales para la detección y clasificación de los diferentes usos del suelo, incluyendo áreas irrigadas.

  • MODELOS DIGITALES DE ELEVACIÓN

    En el proceso de clasificación de las coberturas de uso de suelo es fundamental el utilizar Modelos Digitales de Elevación (MDE). Esto ya que a partir de los MDE podemos obtener las pendientes y a partir de estas se pueden discriminar coberturas de uso de suelo que no nos interesan.

  • Imágenes satelitales Sentinel 2

    Las imágenes satelitales Sentinel 2 son las mas completas ya que cuentan con una resolución espacial de 10 metros y una resolución temporal de 5 días (tomando en cuenta toda la constelación). A partir de ellas podemos crear indices de vegetación como el NDVI o EVI e indices de humedad como el NDWI.

  • Machine Learning (ML)

    Las áreas irrigadas fueron detectadas y clasificadas analizando cada uno de los pixeles de la imagen satelital mediante el algoritmo Random Forest

¿Cómo se realizaron los análisis?

El primer paso fue obtener datos para la entrenamiento y posterior validación del algoritmo. Posteriormente, fue necesario obtener una escena completa del estado de Nuevo León, para eso se analizó un mes completo (julio de 2024). Y mediante los datos de entrenamiento se detectaron las áreas irrigadas de cada uno de los municipios.

  • Se utilizaron 90,609 puntos de entrenamiento.

  • La precisión del modelo fue de 83% (Kappa = 0.83)

  • 64,156 kilómetros cuadrados analizados

  • Se pudo discriminar de manera correcta los indices de vegetación de la zona forestal y las áreas irrigadas.

La importancia del riego de pivote central en la detección de áreas irrigadas

Las tierras agrícolas con riego de pivote central son perfectas para la obtención de puntos de control ya que los valores de reflectancia de estas zonas corresponden de manera exacta con áreas irrigadas. De esta manera eliminamos ruido en los valores de reflectancia que pueda surgir de otras coberturas del suelo.

Factores que influyen al momento de detectar áreas irrigadas mediante imágenes satelitales

Son 4 los factores que mas influyen al momento de detectar áreas irrigadas:

  • Valores de  reflectancia

    Es muy común que coberturas de suelo como el bosque o zonas agrícolas tengan valores de reflectancia similares por lo que a la hora de clasificarlas pueden haber errores. Sin embargo, con los datos de entrenamiento obtenidos a partir de las zonas con riego central este problema fue solucionado.

  • Los datos de entrenamiento deben de ser de calidad

    Los datos de entrenamiento deben de ser de calidad y ademas, los datos deben de ser lo suficientemente grandes para que el algoritmo pueda comprender cada una de las características de cada uso de suelo

  • La altitud sobre el nivel del mar

    En muchas ocasiones cuando se hacen clasificaciones supervisadas no se incluyen valores de altitud. Pero esto es fundamental para la diferenciación de coberturas con valores de reflectancia similares.

  • Mayor entrenamiento no siempre significa que los resultados serán mejores

    Para este caso, se utilizaron 200 arboles de decisión. Con este valor fue con el que obtuvimos mejores resultados.
